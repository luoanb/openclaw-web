## 范围（极简）

**包含**：新增桌面程序 `quick-notes` 的第一阶段交互范围；支持 TODO 进行中队列、完成后归档到默认折叠的已完成区、速记心得/便签；数据保存在本机应用数据目录。

**不包含**：账号体系、云同步、多设备同步、富文本编辑器、复杂标签/搜索、提醒通知、系统托盘、窗口快捷键、数据导入导出。

---

## 功能与交互

### 1. 应用壳

| 项目 | 说明 |
|------|------|
| 主视图 | 单屏桌面工作台，TODO 与速记并列展示；窄窗口下纵向堆叠。 |
| 启动状态 | 启动后读取本地数据；读取失败时展示错误与重试入口。 |
| 数据反馈 | 新增、编辑、完成、删除后自动保存；保存失败时保留当前输入并提示错误。 |
| 空状态 | TODO 和速记均为空时给出轻量引导文案。 |

**验收**

- [ ] 打开应用后能看到 TODO 与速记两个工作区。
- [ ] 本地数据读取失败时不会白屏，并能重试。
- [ ] 空状态能提示用户下一步可做什么。

---

### 2. TODO 队列

| 项目 | 说明 |
|------|------|
| 新增 | 用户输入任务内容后可添加到进行中队列。空内容不可添加。 |
| 展示 | 进行中任务默认展开，按更新时间倒序展示。 |
| 完成 | 用户勾选或点击完成后，任务状态变为 `done`，记录 `completedAt`，从进行中队列移入已完成区。 |
| 已完成 | 已完成区默认折叠，标题显示数量；展开后按完成时间倒序展示。 |
| 恢复 | 已完成任务可恢复到进行中，清空 `completedAt`。 |
| 删除 | 进行中和已完成任务均可删除。 |

**验收**

- [ ] 用户能新增进行中任务。
- [ ] 完成任务后，它不再出现在进行中队列，并出现在已完成区。
- [ ] 已完成区初始为折叠状态，展开后可看到归档任务。
- [ ] 已完成任务可恢复或删除。

---

### 3. 速记

| 项目 | 说明 |
|------|------|
| 新增 | 用户可输入心得、便签等短文本并保存。空内容不可保存。 |
| 展示 | 速记按更新时间倒序展示。 |
| 编辑 | 已有速记可直接编辑并保存更新时间。 |
| 删除 | 用户可删除不再需要的速记。 |

**验收**

- [ ] 用户能新增一条速记。
- [ ] 新增或编辑后，速记列表按更新时间更新排序。
- [ ] 删除速记后不会影响 TODO 数据。

---

## 数据与展示约定

- TODO 展示字段：内容、状态、创建时间、更新时间、完成时间。
- 速记展示字段：内容、创建时间、更新时间。
- 时间字段使用 ISO 字符串，由前端生成后交给 Tauri 保存。
- 首期不做复杂冲突处理；同一窗口内以最新一次操作覆盖保存。
